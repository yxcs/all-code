发布应用

一个应用如果没有用户，它只是一段代码。为了获得用户，您需要发布您的应用。所有 Chrome 应用都通过 Chrome 网上应用店发布。对于不熟悉应用店的人来说，它为用户和开发者提供了许多便利。

对用户来说：

  • 增强的安全性：会检查所有应用和扩展程序中是否包含恶意软件。
  • 所有应用都以开发者发布的最新版本维护。
  • 评分和评论提供了一种很好的方式，以便了解应用的质量。

对于开发者来说：

  • 您只要一次性上传您的应用，它就会发布给所有用户。
  • 内建支付与订阅支持。
  • 您的应用更容易被用户发现。
  • 有一些工具帮您管理用户报告的问题与评论。

Chrome 网上应用店有一个特殊的供开发者使用的信息中心，让您上传新的应用，更新现有的应用。

上传应用的过程很简单：

  1. 将您的应用根目录（包含“manifest.json”文件的文件夹）压缩。
  2. 访问信息中心，并单击“上传新应用”。
  3. 在您的系统中找到该文件。
  4. 上传。

注意：为了尝试发布过程，您可以使用该代码实验室中的任意示例应用，只要确保它包含 manifest.json。

现在在您向所有人发布前，您需要上传关于您的应用的资源和额外的元信息。

  • 应用店中显示的图标
  • 您的应用的详细描述，这会吸引用户下载您的应用。
  • 您的应用的屏幕截图或视频（向用户展现您的应用的外观）
  • 列出您的应用的主要分类
  • 在 Chrome 网上应用店的展区显示的平铺小图标

还有很多其他字段，但是以上这些项目是必须的。

如果您对一切都已满意，您现在可以向公众发布您的应用。如果您还没准备好，您可以保存草稿，以后使用，您也可以向测试员小组发布。
