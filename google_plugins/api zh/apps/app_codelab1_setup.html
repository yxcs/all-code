设置开发环境

依赖环境与前提条件

  • Windows 和 Mac 上的 Chrome Canary 和 Linux 上的 Chromium Dev

  • 打开您喜爱的编辑器。我们正在开发一个用于 Sublime 的 Chrome 应用插件。

设置 Chrome 浏览器

  1. 确保您正在使用 Chrome Canary，访问 chrome://version 并检查第一行末尾是否包含 canary 关键字：

  2. 在 chrome://flags 中启用标志

    • 实验性扩展程序 API
    • 启用打包应用调试

  3. 在 chrome://extensions 中启用开发者模式：

注意：请在单独的窗口打开 chrome://extensions 网页，并在您进行实验的过程中使它随时可以访问，您将会多次使用该网页加载和重新加载“待办事项”应用。
