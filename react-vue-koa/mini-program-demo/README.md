> A weui-lan project

![weui-lan](https://github.com/lanzhsh/react-vue-koa/blob/master/mini-program-demo/images/weui-lan.jpg)

## instructions 

- 里面五个类型组件:
- 1.音频:
- 常见音频,背景音频,录音
- 2.文件:
- 依赖weui的上传图片组件
- 3.位置
- 获取所在的经纬度
- 4.地图
- 小程序自带地图
- 5.画布
- 里面使用wxa-plugin-canvas插件生成海报

## 使用方法
- git-clone
- 在微信开发者工具打开,找到对应功能的组件复制即可
